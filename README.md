## 📝 Descripción

Este repositorio constituye el **Portafolio de Actividades** para la asignatura de **Programación Orientada a Objetos I**. El objetivo central es documentar y gestionar el progreso académico en el dominio del paradigma de la POO utilizando el lenguaje **Java**.

Este espacio funciona como un **Monorepositorio**, donde cada actividad se organiza en su propia subcarpeta para mantener la independencia de los proyectos educativos sin perder la trazabilidad del historial global.

## 🧰Tecnologías utilizadas

**Lenguaje:** Java JDK 21

**Herramientas:** NetBeans / VS Code / Git Bash

**Gestor de dependencias:** Maven / Ant

**Control de versiones:** Git & GitHub
